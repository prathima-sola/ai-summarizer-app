# AI Text Summarizer 🤖

A full-stack AI application that summarizes any text using Claude AI.

## 🔗 Live Demo
[https://ai-summarizer-app-zeta.vercel.app](https://ai-summarizer-app-zeta.vercel.app)

## 🛠 Tech Stack
- **Frontend:** React, deployed on Vercel
- **Backend:** Node.js, Express, deployed on Render
- **AI:** Anthropic Claude API
- **DevOps:** Docker, GitHub Actions CI/CD

## ✨ Features
- Paste any text and get an instant AI summary
- 5 clear bullet points generated by Claude AI
- Containerized with Docker
- Automated CI/CD pipeline with GitHub Actions

## 🚀 Run Locally
git clone https://github.com/prathima-sola/ai-summarizer-app
cd ai-summarizer-app
docker-compose up --build
